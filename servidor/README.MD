# Bibliotecas utilizadas

- "bcryptjs": encrptar password
- "cookie-parser" : enviar cookies en api
- "cors": permite conectar back con front
- "dotenv" : Mantener seguras las variables de entorno
- "express" : permite crear la api
- "jsonwebtoken" : Token para autenticacion
- "node-postgres": conectar con postgres
- "nodemailer" : Enviar email a clientes
- "nodemon" : Actualizacion del servidor
